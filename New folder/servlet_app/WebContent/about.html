Objective

Creating and managing Audit Controls electronically. The system will help the Bank in reducing the cycle time in closing the Audit Irregularities / non-compliance of various accounts. The system will be a web portal where the Audit Executives would enter compliance / discrepancy details for each application form. The Branch Managers would get the Audit report online, follow up the sales Executive to comply with the set rules and close the audit Irregularities online. The system will help the management in assessing the performance of Sales Executives, Branch Managers and Audit Executives with the help of various Analytical reports. The system will also help the management in awarding Internal Audit grades to the Branches.



goal of site

System allows the registered user to login and provide role based access to system features / functionality.
Executive can create, update, delete various Audit norms for different types of accounts in the system and system maintains centralized data of it.
System allows the management executive to create, update, delete rules for calculating the Internal Audit grade to Branch and maintains centralized data of it.
System allows the Sales Executives to enter the new account details opened by the customer with the Bank. The sales executive details are stored for each new account opened to keep track of sales made by the sales Executive and also for responsibility for meeting all the required audit compliance norms for the account.
Audit Executives enter discrepancy / compliance details for each application submitted Branch Manager gets Sales Executive wise discrepancy report and follow up the respective sales Executive for compliance.
Sales Executive closes discrepancies by entering required compliance details. System changes the status of each norm for each application and up-dates the database
System sends auto mailers to Branch Manager, if discrepancy not closed within specified time limit by Sales Executive
System generates various analytical report
